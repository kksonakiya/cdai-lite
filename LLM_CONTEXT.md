
# 🧠 CDAI Lite — Context File

---

# 🔥 Product Identity

CDAI Lite is NOT a generic AI image generator.

It is:

> **An AI Design Decision Tool for creators**

Core idea:

> Generate multiple variations → compare them → pick the best one

---

# 🎯 Target Audience (ICP)

Primary users:

- Etsy sellers
- Print-on-demand (POD) creators
- Digital product creators

Secondary users:

- AI artists experimenting with styles
- Developers exploring model behavior

---

# 🚨 Core Problem

Creators today:

- Generate designs using AI tools (Midjourney, Stable Diffusion, etc.)
- Manually review outputs
- Iterate one-by-one (slow and inefficient)
- Spend money on credits/subscriptions
- Guess which design will perform well
- Upload and hope

There is NO structured way to:

> Compare multiple variations and confidently select the best design

---

# 💡 Solution

CDAI Lite enables:

> **Structured experimentation for design creation**

Workflow:

1. Run batch experiments (multiple prompts, models, LoRAs)
2. Let jobs run in background
3. Compare outputs side-by-side
4. Identify best designs
5. Use/export the winner

---

# ⚙️ Core Capabilities

- Checkpoint comparison
- LoRA comparison
- Prompt comparison
- Batch generation (background jobs)
- Local GPU/CPU rendering
- Gallery view for visual comparison
- Prompt view for structured evaluation
- Image upscaling (up to 4x)
