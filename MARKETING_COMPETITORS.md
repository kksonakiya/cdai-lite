# ⚔️ CDAI Lite — Competitor Context

## 🧠 Category Definition

CDAI Lite operates in a **new category**:

> AI Design Decision Tools

Most competitors do NOT operate in this category directly.

Instead, they solve **parts of the workflow**.

---

# 🧩 Current User Workflow (Reality)

Typical Etsy creator workflow:

1. Generate images → Midjourney / Stable Diffusion
2. Edit / adjust → Canva / Kittl
3. Create product → Printify / Printful
4. Optimize listing → eRank / Marmalead
5. Upload and hope

❗ Missing step:

> No structured way to compare and select the best design

---

# 🟢 Category 1 — AI Image Generators

## Examples:

- Midjourney
- Leonardo AI
- Stable Diffusion UIs

### Strengths:

- High-quality image generation
- Fast iteration
- Popular among creators

### Weaknesses:

- Credit-based or subscription-limited
- No structured comparison system
- No batch experimentation workflow
- Users must manually iterate (prompt → result → retry)

### Takeaway:

> They generate options, but do not help choose the best one

---

# 🔵 Category 2 — Design Tools

## Examples:

- Canva
- Kittl
- Adobe Express

### Strengths:

- Easy to use
- Templates and mockups
- Some AI generation features

### Weaknesses:

- Limited experimentation capability
- No batch comparison
- Not built for testing variations
- Often subscription-based

### Takeaway:

> They help refine designs, not test them at scale

---
