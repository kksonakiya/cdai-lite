# ✅ KEYWORD VALIDATION CHECKLIST (MANDATORY)

### 🔍 Keyword Selection

* Must have **≥100 monthly search volume** (GKP validated)
* Prefer **100–1K or 1K–10K range**
* Must be **Etsy / design / POD relevant**
* Must NOT be purely buyer-only (e.g., “buy t-shirt”)

---

### 🎯 Intent Check

* Keyword must match at least one:
* Creator intent (sell, design, create)
* Problem intent (ideas, why not selling, improve)
* Reject keywords with **pure browsing intent only**

---

### 💰 CPC Validation (Commercial Intent)

* Must have **non-zero top-of-page bid**
* Prefer **₹20+ low bid OR ₹100+ high bid**
* If CPC = ₹0.94 → treat as **low priority**

---

### 📊 Competition Reality Check

* Search keyword on YouTube / Pinterest
* Confirm:
* Videos/pins exist (demand proven)
* NOT dominated by huge channels only
* Reject if:
* No content exists (no demand)
* OR only mega channels dominate

---

### 🎥 Content Feasibility Check

* Can you create **visual output** for this keyword?
* Can you show:
* comparison?
* variation?
* result?
* Reject if:
* cannot demonstrate visually

---

### 🔁 Variation Potential

* Can this keyword produce **10+ content variations?**
* Example:
* “t-shirt design ideas” → YES
* “etsy logo png exact” → NO

---

### 📈 Traffic Expectation Logic

* Estimate:
* 100–1K keyword → expect 20–50 views
* 1K–10K keyword → expect 50–300 views
* If expectation = near zero → reject keyword

---

### 📌 Pinterest Fit Check

* Keyword must map to:
* aesthetic / idea / inspiration
* Reject if:
* too technical (e.g., “setup python sd”)

---

### 🧪 YouTube Fit Check

* Can this keyword be turned into:
* a hook?
* a problem statement?
* Reject if:
* cannot form a compelling title

---

### 🔥 Final Go / No-Go Rule

* If keyword passes **ALL checks → CREATE**
* If fails **ANY 2 checks → REJECT**

---

# 🚨 NON-NEGOTIABLE RULE

> ❗ No keyword → No content
>
> ❗ No data → No execution
